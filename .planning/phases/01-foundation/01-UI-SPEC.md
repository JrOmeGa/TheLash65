# Phase 1 — UI Design Contract

> Visual and interaction contract for frontend phases. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | shadcn/ui (CLI 4.1.0, components copied into project) |
| Preset | Champagne Minimalist — "The Ethereal Atelier" |
| Component library | Radix UI (via shadcn) |
| Icon library | Material Symbols Outlined, weight 100–300 (thin) |
| Heading font | Noto Serif (next/font/google) |
| Body font | Manrope (next/font/google) |
| Thai fallback font | Sarabun (next/font/google) |

### Font Loading

All three fonts are loaded via `next/font/google` in `app/[locale]/layout.tsx`. Do NOT use `<link>` tags to Google Fonts.

```ts
// app/[locale]/layout.tsx
import { Noto_Serif, Manrope, Sarabun } from 'next/font/google'

const notoSerif = Noto_Serif({
  subsets: ['latin'],
  weight: ['400', '600'],
  variable: '--font-noto-serif',
  display: 'swap',
})

const manrope = Manrope({
  subsets: ['latin'],
  weight: ['400', '600'],
  variable: '--font-manrope',
  display: 'swap',
})

const sarabun = Sarabun({
  subsets: ['thai', 'latin'],
  weight: ['400', '600'],
  variable: '--font-sarabun',
  display: 'swap',
})
```

Apply variables to `<html>`: `className={`${notoSerif.variable} ${manrope.variable} ${sarabun.variable}`}`.

CSS font stacks (in global CSS or Tailwind v4 theme):

```css
/* Headings */
font-family: var(--font-noto-serif), var(--font-sarabun), serif;

/* Body / UI labels */
font-family: var(--font-manrope), var(--font-sarabun), sans-serif;
```

Sarabun covers all Thai glyphs for both roles. It activates automatically when the browser encounters Thai characters not covered by Noto Serif or Manrope.

## Spacing Scale

Declared values (all multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-label gaps, inline badge padding |
| sm | 8px | Button inner padding (vertical), tight list gaps |
| md | 16px | Default element spacing, card inner padding |
| lg | 24px | Section inner padding, card-to-card gap |
| xl | 32px | Layout column gap, hero content padding |
| 2xl | 48px | Major section breaks between page sections |
| 3xl | 64px | Page-top and page-bottom whitespace |

Exceptions:

- Touch targets (nav icons, toggle, CTA buttons): minimum 44px height — enforced via `min-h-[44px]` on all tappable elements.
- Sticky Mandate Bar pill: 52px height — intentionally exceeds standard scale for prominence.
- Gallery gutter: 8px (`sm`) on mobile, 12px on desktop — 12px is an intentional off-scale value for tighter editorial image grids.

## Typography

### Scale

Exactly 4 sizes. Display (48px) is deferred — Phase 1 has no hero headline large enough to warrant it.

| Role | Size | Font | Weight | Line Height | Tracking | Usage |
|------|------|------|--------|-------------|----------|-------|
| Heading | 30px / 1.875rem | Noto Serif | 600 | 1.1 | -0.02em | Section headers (Portfolio, Services, About) |
| Card Heading | 20px / 1.25rem | Noto Serif | 600 | 1.2 | -0.02em | Service card names |
| Body | 14px / 0.875rem | Manrope | 400 | 1.5 | 0 | Paragraphs, card descriptions, bio |
| Label / Micro | 12px / 0.75rem | Manrope | 400 or 600 | 1.4 | 0 or 0.2em | Price, duration, nav items (weight 400, no tracking); all-caps eyebrows and category tags (weight 600, `text-transform: uppercase`, `letter-spacing: 0.2em`) |

Rules:

- Heading and Card Heading use Noto Serif. Body and Label/Micro use Manrope.
- Exactly 2 weights: 400 (regular) for body and standard labels; 600 (semibold) for all headings and all-caps micro-labels. Weight 700 is not used anywhere.
- Micro-label visual distinction (eyebrows, category tags) is achieved via `text-transform: uppercase` and `letter-spacing: 0.2em` at 12px — no separate font size needed.
- Italics are permitted in Heading and Card Heading roles for editorial emphasis (e.g., `<em>Every Lash</em>`). Do not use italics in Body or Label.
- Never use pure `#000000` for text. All text uses `#1F1B18` (on-surface) or its opacity variants.
- Body text is left-aligned. Center-align is reserved for brief standalone value propositions (hero sub-headline only).

## Color

### Palette

| Role | Hex / Value | Usage |
|------|-------------|-------|
| Surface (60%) | `#fff8f5` | Page background, base canvas |
| Surface container low (30%) | `#f5ede6` | Section alternating backgrounds, footer |
| Surface container lowest | `#ffffff` | Floating cards, lightbox backdrop — "pop" layer |
| Accent (10%) | `#755944` | See reserved list below |
| On-surface (text) | `#1F1B18` | All body text, headings, labels |
| Destructive | `#b91c1c` | Destructive actions only — Phase 1 has none |

Accent `#755944` is reserved exclusively for:

1. Primary CTA button fill (linear gradient `#755944` → `#9c7660`)
2. Sticky Mandate Bar background
3. Active/selected state indicator on language toggle
4. Price text on service cards
5. Hover underline on navigation links

Accent is NOT used for: borders, dividers, icon fills, backgrounds, section separators.

### The "No-Line" Rule

Do not use `1px solid` borders for sectioning or card containment. Section separation is achieved exclusively via:

- Background color shift between `#fff8f5` (surface) and `#f5ede6` (surface-container-low)
- Editorial ambient shadow: `box-shadow: 0 10px 40px rgba(31, 27, 24, 0.06)`

The only permitted `border` property is `border: 1px solid rgba(255, 255, 255, 0.2)` on glassmorphic floating elements (nav bar, sticky bar).

### Glass & Gradient Rule

All floating UI (top nav bar, sticky Mandate Bar) must apply:

```css
background: rgba(255, 248, 245, 0.80); /* surface at 80% opacity */
backdrop-filter: blur(12px);
-webkit-backdrop-filter: blur(12px);
border: 1px solid rgba(255, 255, 255, 0.2);
```

`backdrop-blur` must be at least 12px. Do not reduce below 12px for any floating element.

## Component Specifications

### Focal Points

- **Mobile primary focal point:** The Sticky Mandate Bar. It is the highest-elevation element on mobile and the first action target the eye lands on after scanning the hero.
- **Desktop primary focal point:** The Portfolio Gallery grid. The 3-column image grid anchors the above-the-fold experience on wider viewports.

### Navigation Bar

- Fixed to top on all pages.
- Height: 56px (14px × 4 scale unit).
- Layout: Logo left | Page links center (desktop only) | "TH | EN" toggle + Book CTA right.
- Glassmorphic treatment: surface 80% + `backdrop-blur-[12px]` + `border: 1px solid rgba(255,255,255,0.2)`.
- Mobile: hide page links. Show logo left, language toggle + hamburger icon right.
- Mobile menu: shadcn Sheet (bottom drawer), full height, same glassmorphic surface.
- Hamburger icon button: `aria-label="Open menu"` (English) / `aria-label="เปิดเมนู"` (Thai — driven by current locale). `min-h-[44px] min-w-[44px]`.
- Nav link style: Manrope 12px Label weight 400. Active page: accent underline 1px, color `#755944`.
- All nav tap targets: `min-h-[44px] min-w-[44px]`.

### Language Toggle

- Placement: top-right of nav bar, inline with CTA.
- Markup: `<button>TH</button><span aria-hidden>|</span><button>EN</button>`.
- Active locale: weight 600, color `#755944`.
- Inactive locale: weight 400, color `#1F1B18` at 50% opacity.
- On toggle: navigate to `/{newLocale}{currentPath}` using next-intl `useRouter().replace()` + `usePathname()`.
- No animation. Instant locale switch.

### Primary CTA Button (Book / นัดหมาย)

- Label: "นัดหมาย" (Thai) | "Book an Appointment" (English).
- Shape: `border-radius: 0.125rem` (2px — near-rectangular).
- Fill: `linear-gradient(135deg, #755944 0%, #9c7660 100%)`.
- Text: Manrope 14px Body, weight 600, color `#ffffff`.
- Padding: 12px 24px (sm top/bottom, lg left/right).
- Active state: `active:scale-[0.98]` with `transition-transform duration-100`.
- No border, no shadow on the button itself.

### Sticky Mandate Bar

- A floating pill anchored to the bottom of the viewport on mobile only (hidden on desktop ≥768px).
- Height: 52px, `border-radius: 26px` (pill shape).
- Background: `#755944` (accent, full opacity — high contrast, no blur).
- Label: left side, Manrope 14px Body weight 400, color `#fff8f5`.
- Action button: right side, circular 36px, `background: #ffffff`, icon in `#755944`.
- Shadow: `0 10px 40px rgba(31, 27, 24, 0.18)` (stronger than ambient, elevated element).
- Phase 1: bar is visible with label "จองคิวออนไลน์" / "Book Online" but button is non-functional (links to placeholder).

### Portfolio Gallery

- Grid: 2 columns on mobile (`grid-cols-2`), 3 columns on tablet/desktop (`md:grid-cols-3`).
- Gap: 8px mobile, 12px desktop.
- Image aspect ratio: 4:5 (`aspect-[4/5]`) — all gallery images must use this ratio. 3:4 (`aspect-[3/4]`) permitted for portrait-heavy shots. Never 16:9.
- Image treatment: `border-radius: 0.75rem` (12px) on each image tile.
- Loading: `<Image loading="lazy" />` — no skeleton, no blur placeholder for v1.
- Lightbox: shadcn Dialog component. On tile tap, open Dialog containing the image at full available width, `object-fit: contain`, black backdrop (`bg-black/90`). Close button: top-right corner, Material Symbols `close` icon, white, 44×44px tap target, `aria-label="Close"` (English) / `aria-label="ปิด"` (Thai — driven by current locale).
- Lightbox does not show caption, navigation arrows, or counter in Phase 1.

### Service Cards

- Layout: full-width stack on mobile, 2-column grid on desktop (`md:grid-cols-2`), gap `lg` (24px).
- Card base: `background: #ffffff` (surface-container-lowest), `border-radius: 0.75rem`, editorial shadow `0 10px 40px rgba(31, 27, 24, 0.06)`. No border.
- Card padding: 24px (`lg`) on all sides.
- Card content order (top to bottom):
  1. Micro label: service category tag, 12px Manrope weight 600, `text-transform: uppercase`, `letter-spacing: 0.2em`, color `#755944` at 60% opacity.
  2. Service name: Noto Serif 20px Card Heading weight 600, color `#1F1B18`, line-height 1.2, tracking -0.02em.
  3. Description: Manrope 14px Body weight 400, color `#1F1B18` at 75% opacity, line-height 1.5.
  4. Divider: 16px vertical space (`md`) — no line, just whitespace.
  5. Duration + Price row: flex row, space-between.
     - Duration: Material Symbols `schedule` icon (16px) + Manrope 12px Label weight 400, color `#1F1B18` at 60%.
     - Price: Manrope 14px Body weight 600, color `#755944`. Format: `฿{number}` — no decimals, no "from".
- Active state: `active:scale-[0.98]` on card (Phase 3 will add tap-to-select behavior).
- Empty state (no services in DB): render a single full-width card with centered copy — see Copywriting Contract.

### About Page

- Layout: single column, max-width 640px, centered on desktop.
- Owner photo: `aspect-[4/5]`, `border-radius: 0.75rem`, 100% width on mobile, 320px width on desktop, float or editorial overlap at discretion of executor.
- Bio: Manrope 14px Body weight 400, line-height 1.5, left-aligned.
- Contact block: two rows. Each row: Material Symbols icon (20px, `#755944`) + Label 12px Manrope weight 400. Line ID row uses `call_to_action` icon. Phone row uses `call` icon.
- Shop hours: static text block. Heading "Hours" / "เวลาทำการ" in Manrope 12px, weight 600, `text-transform: uppercase`, `letter-spacing: 0.2em`. Hours in Manrope 14px Body weight 400. No live status indicator.
- CTA button: full-width on mobile, auto-width on desktop. Same style as Primary CTA. Phase 1: links to `#` placeholder.

### Footer

- Background: `#f5ede6` (surface-container-low).
- Single row on desktop, stacked on mobile.
- Content: Logo left | copyright center | social/contact icons right (Line, Phone).
- Copyright: Manrope 12px Label weight 400, color `#1F1B18` at 50% opacity.
- No border-top. Transition from page surface to footer via background color shift only.

## Interaction Patterns

### Universal Active State

Every tappable element applies `active:scale-[0.98]` with `transition: transform 100ms ease-out`. This is not optional — it is the tactile feedback contract for the entire design system. No exceptions.

### Scroll Behavior

- `scroll-behavior: smooth` globally.
- Nav bar remains fixed; page content scrolls underneath.
- No scroll-triggered animations in Phase 1 (deferred to Phase 5 if at all).

### Hover States (Desktop)

- Nav links: accent (`#755944`) underline slides in from left, `transition: width 200ms ease`.
- Service cards: ambient shadow deepens — `0 16px 48px rgba(31, 27, 24, 0.10)`. No scale.
- Gallery tiles: subtle scale `hover:scale-[1.02]`, `transition: transform 200ms ease`.
- Buttons: 8% brightness increase via `hover:brightness-[1.08]`. No outline, no color change.

### Focus States (Accessibility)

- All interactive elements: `focus-visible:outline-2 focus-visible:outline-offset-2 focus-visible:outline-[#755944]`.
- Use `focus-visible` (not `focus`) to avoid outline on mouse click.

### Language Toggle Transition

- No page transition animation. Next.js navigation replaces URL; content swaps instantly.
- The toggle updates synchronously — no loading indicator.

## Date Formatting Contract

All date rendering must use `next-intl`'s `useFormatter()` hook with explicit Gregorian calendar override.

```ts
// Correct pattern — prevents Buddhist Era (BE) offset in Thai locale
const format = useFormatter()
format.dateTime(date, {
  calendar: 'gregory',
  year: 'numeric',
  month: 'long',
  day: 'numeric',
})
```

Never use `new Date().toLocaleDateString('th-TH')` without `{ calendar: 'gregory' }` — this returns Buddhist Era years (2568 instead of 2025) and is a known bug.
