   - `users` table: `id` (text PK, Better Auth compatible), `name` (text, notNull), `email` (text, notNull, unique), `emailVerified` (boolean, default false), `image` (text nullable), `createdAt`/`updatedAt` timestamps

   - `bookingStatusEnum`: pgEnum with values `['pending', 'confirmed', 'cancelled']`
   - `bookings` table: `id` (uuid, defaultRandom PK), `userId` (text, FK to users.id), `serviceId` (integer, FK to services.id), `scheduledAt` (timestamp with timezone, notNull), `status` (bookingStatusEnum, default 'pending'), `paymentConfirmedAt` (timestamp with timezone nullable), `notes` (text nullable), timestamps. Indexes on `userId` and `scheduledAt`.
   - `scheduleRules` table: `id` (integer, identity PK), `dayOfWeek` (integer 0-6, notNull), `openTime` (text "HH:MM", notNull), `closeTime` (text "HH:MM", notNull), `slotDurationMinutes` (integer, default 120), `isActive` (boolean, default true), timestamps
   - `scheduleExceptions` table: `id` (integer, identity PK), `exceptionDate` (timestamp with timezone, notNull), `isClosed` (boolean, default true), `reason` (text nullable), timestamps

   All timestamp columns use `timestamp('column_name').defaultNow().notNull()` for createdAt/updatedAt. Per D-19, all timestamps stored as UTC (Drizzle default behavior with `timestamp` type).

   /**
    * Format a date for display, forcing Gregorian calendar to prevent
    * Buddhist Era years in Thai locale (th-TH defaults to BE+543).
    */
   export function formatDateGregorian(
     date: Date,
     locale: string,
     options?: Intl.DateTimeFormatOptions,
   ): string {
     return new Intl.DateTimeFormat(locale, {
       calendar: 'gregory',
       year: 'numeric',
       month: 'long',
       day: 'numeric',
       ...options,
     }).format(date);
   }

8. Create `tests/unit/date-format.test.ts` (I18N-03 — D-18):
   ```typescript
   import { describe, it, expect } from 'vitest';
   import { formatDateGregorian } from '@/lib/date-utils';

   describe('Date formatting - Gregorian calendar guard (I18N-03)', () => {
     const testDate = new Date('2025-06-15T12:00:00Z');

     it('formats Thai locale date with Gregorian year (not Buddhist Era)', () => {
       const result = formatDateGregorian(testDate, 'th-TH');
       // Buddhist Era would show 2568 instead of 2025
       expect(result).toContain('2025');
       expect(result).not.toContain('2568');
     });

     it('formats English locale date with correct year', () => {
       const result = formatDateGregorian(testDate, 'en-US');
       expect(result).toContain('2025');
     });

     it('includes month and day in output', () => {
       const result = formatDateGregorian(testDate, 'en-US');
       expect(result).toContain('June');
       expect(result).toContain('15');
     });

     it('accepts custom format options while preserving Gregorian calendar', () => {
       const result = formatDateGregorian(testDate, 'th-TH', {
         year: 'numeric',
         month: 'short',
         day: 'numeric',
       });
       expect(result).toContain('2025');
       expect(result).not.toContain('2568');
     });
   });
   ```
